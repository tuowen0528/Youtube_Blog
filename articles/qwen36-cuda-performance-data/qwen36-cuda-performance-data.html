05 · Recommendation

本机测试的最佳参数方向

结论针对 RTX 3060 12GB、llama.cpp b9672、IQ2_M 主模型和 F16 mmproj。 不同显卡、构建版本及后台显存占用可能得到不同结果。

最高速度配置

使用全部 GPU 层、关闭自动适配、保留 Q8 KV、把服务 slot 固定为 1， 上下文设置为 8192。视觉任务偏向低延迟时，将单图视觉 Token 固定为 1024。

显存余量很小： 最快配置测试时整卡占用 11873 MiB，理论只剩约 415 MiB。 如果后台运行 NVIDIA Broadcast、Wallpaper Engine、视频应用、GPU Overlay 或重度浏览器页面，可能出现速度骤降、共享显存分页或 OOM。 不稳定时应改用 -ngl auto -fitt 256 的保守配置。

数据来源：本机调优报告《llama.cpp CUDA 视觉模型性能排查与调优报告》。 llama-bench、短测、服务端 API 长测和真实图片请求属于不同负载， 不能把所有数值视为完全相同的基准。参数选择应优先参考 384 Token 服务端长测和真实视觉请求。
